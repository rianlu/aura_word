# WordFlow (词流)

**WordFlow (词流)** 是一款专为初中生打造的英语同步智能练习站。针对 2024/2025 外研版（FLTRP）新教材难度大幅提升的挑战，我们提供了一套更硬核、更高效的“听说写”闭环训练系统。

## 核心价值：为什么选择 WordFlow？

*   **深度同步新教材**：紧贴县城初中教学进度，实时适配 2024/2025 秋季新版外研社教材单词库。
*   **三位一体训练流**：拒绝“混个眼熟”。每个单词必须经过“选义（视觉辨析）”、“口语（语音输入）”、“拼写（记忆输出）”三个环节，形成完整的记忆闭环。
*   **科学进度控制**：内置 SM-2 间隔重复算法，根据记忆遗忘曲线自动安排复习，确保高强度练习下的长效留存。
*   **解放家长（开发中）**：即将上线“全自动同步听写”功能，彻底解决家长无法辅助听写的痛点。

## 产品特性

*   **极致视觉体验**：采用极简主义设计，纯白背景配合高饱和度活力配色，减少视觉干扰，让注意力专注在单词本身。
*   **智能发音系统**：优先请求有道/Google 真人发音 API，并支持本地缓存，确保音质地道且节省流量。
*   **安全数据备份**：支持 AES 加密的 `.wcc` 格式导出，学习进度跨设备安全迁移。
